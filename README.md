\# ✈️ Flight Deal Tracker



This is a Python automation project that finds cheap flight deals and sends notifications when prices drop below a set threshold.



\---



\## 🚀 Features



\- Fetches flight data from API

\- Detects cheapest flights (direct \& indirect)

\- Compares prices with Google Sheet data

\- Sends SMS alerts using Twilio

\- Sends email notifications to users

\- Stores customer data in Google Sheets (via Sheety)



\---



\## 🛠️ Tech Stack



\- Python

\- Requests

\- Sheety API (Google Sheets)

\- Amadeus / Flight Search API

\- Twilio (SMS)

\- SMTP (Email)

\- dotenv (environment variables)



\---



\## 📁 Project Structure



\- `main.py` → Main program logic

\- `flight\_search.py` → Flight API handling

\- `flight\_data.py` → Flight processing

\- `data\_manager.py` → Google Sheets integration

\- `notification\_manager.py` → SMS \& Email alerts
